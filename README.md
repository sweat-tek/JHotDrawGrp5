# jhotdraw

* maven build process

## Getting started
Build the project by running `mvn clean compile` in the root directory  
Run the application from the main class in `jhotdraw-samples/jhotdraw-samples-misc/src/main/java/org/jhotdraw/samples/svg/Main.java`
